## Procedimiento de instalación Garfield++ local en Ubuntu:

1. Pre-requisitos:

	sudo apt update	
	sudo apt install build-essential 
	sudo apt install gfortran        

	sudo apt install libgsl-dev     
	sudo apt install libtbb12
	sudo apt install libvdt-dev

	sudo apt install cmake
	sudo apt install clang	
		
	descargar .tar.gz de la distro indicada en https://root.cern.ch/releases/release-64000/
	
	# Los siguientes pasos hacerlo en la carpeta donde este el archivo .tar.gz, recomedable ~/
	tar -xzvf root_v6.40.00.Linux-ubuntu24.04-x86_64-gcc13.3.tar.gz 
	
	# añadir el siguiente comando en ~/.bashrc (no olvides ejecutar)
	source $HOME/root/bin/thisroot.sh 
	
2. Instalación Garfield++:

	# añadir al ~/.bashrc (no olvides ejecutarlo)
	export GARFIELD_HOME=$HOME/garfield

	git clone https://gitlab.cern.ch/garfield/garfieldpp.git $GARFIELD_HOME

	cd $GARFIELD_HOME
	mkdir build
	cd build
	cmake $GARFIELD_HOME
	# nproc se refiere al numero de cores de tu pc
	make -j$(nproc)  
	make install
	
	# añadir al ~/.bashrc (no olvides ejecutar)
	source $GARFIELD_HOME/install/share/Garfield/setupGarfield.sh

3. Instalación Elmer (Simulación FEM):
	
	# Agregas ppa al apt que contiene el proyecto Elmer
	sudo add-apt-repository ppa:elmer-csc-ubuntu/elmer-csc-ppa
	sudo apt update
	sudo apt install elmerfem-csc-eg
	sudo apt install gmsh

### Procedimiento Elmer --> Computo del campo electricto

# Desarrollo Drift Tube --> Ruta: ProyectoFiscomp1/scr/DriftTube_prueba/

1. Generación de la malla empleando la libreria gmsh:

	# Esta libreria permite definir la  geometria y crea el mallado
	# ver la estructura dentro de FEM/malla_dt.py 
	# gmsh permite emplearse desde python por lo que nos facilita la sintaxis

	# Este script se puede correr desde cualquier ubicación relativa
	python3 malla_dt.py  
	
	# el output es una archivo --> drift_tube.msh
	# convertir el msh en la info que necesita Elmer
	cd TU_RUTA/ProyectoFiscomp1/scr/DriftTube_prueba/FEM
	ElmerGrid 14 2 drift_tube.msh -autoclean

2. Solución del Campo empleando Elmer
	
	# La ultima linea ElmerGrid genera una carpeta llamada drift_tube

	cd drift_tube
	nano dielectrics.dat 

	# En el dat pegaremos las siguientes lineas

	1 1.0
	
	# Ctrl + o para guardar , Ctrl + x para cerrar

	cd ..
	
	# ejecutar script de Elmer, el script de Elmer lee la malla y asigna las
	# instruciones sobre que ecuación y como se va a resolver
	
	ElmerSolver case.sif
	
	# La solución se entrega como drift_tube/case_t0001.vtu
